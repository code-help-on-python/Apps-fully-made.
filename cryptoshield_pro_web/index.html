Format details

This page supports CryptoShield text tokens. Format: base64url( "CSP1" + 16-byte salt + Fernet token). Key derivation: PBKDF2-HMAC-SHA256 (200,000 iterations). Encryption: Fernet (AES-128-CBC + HMAC-SHA256).
